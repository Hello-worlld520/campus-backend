\1. 解压这个 ZIP 到桌面或其他地方，文件夹名叫 campus_backend

\2. 打开 Anaconda Prompt（开始菜单搜索 Anaconda Prompt）

\3. 激活你的 conda 环境（如果环境名叫 campus，就敲下面这行；如果名字不同，改成你的）：   conda activate campus

   如果没有 campus 环境，先创建：   conda create -n campus python=3.10   conda activate campus

\4. 进入项目文件夹：   cd Desktop\campus_backend

\5. 安装依赖：   pip install -r requirements.txt

\6. 启动后端：   python app.py

\7. 启动成功后（命令行出现绿色 INFO 信息），打开浏览器输入：   http://127.0.0.1:8000

   看到 {"message": "后端启动成功！..."} 就 OK 了！

\8. 测试上传视频：   浏览器打开 http://127.0.0.1:8000/docs   点 /analyze → Try it out → Choose File 选个 mp4 视频上传 → Execute   会返回假的分析结果

\9. 关掉窗口（Ctrl+C）就能停止服务。
