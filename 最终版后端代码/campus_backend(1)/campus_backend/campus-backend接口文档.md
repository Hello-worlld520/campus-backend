# campus-backend接口文档

## 接口总览

| 序号 | 接口            | 方法 | 说明                      |
| :--- | :-------------- | :--- | :------------------------ |
| 1    | `/`             | GET  | 获取API信息               |
| 2    | `/status`       | GET  | 获取系统状态              |
| 3    | `/video_feed`   | GET  | 获取视频流（摄像头/RTSP） |
| 4    | `/upload_video` | POST | 上传本地视频文件          |
| 5    | `/stop`         | GET  | 停止指定视频流            |
| 6    | `/stop_all`     | GET  | 停止所有视频流            |

------

## 接口1：获取API信息

| 项目 | 内容                      |
| :--- | :------------------------ |
| 接口 | `GET /`                   |
| 描述 | 获取API基本信息和使用说明 |

**响应参数：**

| 参数         | 类型    | 说明             |
| :----------- | :------ | :--------------- |
| name         | string  | 项目名称         |
| version      | string  | 版本号           |
| status       | string  | 运行状态         |
| model_loaded | boolean | 模型是否加载成功 |
| endpoints    | object  | 可用接口列表     |



------

## 接口2：获取系统状态

| 项目 | 内容             |
| :--- | :--------------- |
| 接口 | `GET /status`    |
| 描述 | 获取系统运行状态 |

**响应参数：**

| 参数           | 类型    | 说明                 |
| :------------- | :------ | :------------------- |
| status         | string  | 系统状态（ok/error） |
| model_loaded   | boolean | 模型是否加载成功     |
| active_streams | int     | 当前活跃的视频流数量 |
| active_list    | array   | 活跃流ID列表         |



------

## 接口3：获取视频流

| 项目     | 内容                        |
| :------- | :-------------------------- |
| 接口     | `GET /video_feed`           |
| 描述     | 获取实时视频流（MJPEG格式） |
| 返回类型 | multipart/x-mixed-replace   |

**请求参数：**

| 参数   | 类型   | 必填 | 说明                    |
| :----- | :----- | :--- | :---------------------- |
| source | string | 是   | 视频源地址（默认值为0） |

**参数说明：**

| 参数值      | 说明           |
| :---------- | :------------- |
| 0           | 默认摄像头     |
| 1           | 外接摄像头     |
| rtsp://地址 | RTSP网络摄像头 |

**响应：** MJPEG视频流

------

## 接口4：上传视频文件

| 项目         | 内容                       |
| :----------- | :------------------------- |
| 接口         | `POST /upload_video`       |
| 描述         | 上传本地视频文件进行AI分析 |
| Content-Type | multipart/form-data        |

**请求参数：**

| 参数  | 类型 | 必填 | 说明     |
| :---- | :--- | :--- | :------- |
| video | file | 是   | 视频文件 |

**支持的文件格式：** mp4、avi、mov

**响应：** MJPEG视频流

**错误响应：**

| 状态码 | 说明                     |
| :----- | :----------------------- |
| 400    | 没有上传文件或文件名为空 |

------

## 接口5：停止指定视频流

| 项目 | 内容             |
| :--- | :--------------- |
| 接口 | `GET /stop`      |
| 描述 | 停止指定的视频流 |

**请求参数：**

| 参数      | 类型   | 必填 | 说明     |
| :-------- | :----- | :--- | :------- |
| source_id | string | 是   | 视频流ID |

**响应参数：**

| 参数    | 类型   | 说明         |
| :------ | :----- | :----------- |
| message | string | 操作结果信息 |

**错误响应：**

| 状态码 | 说明                |
| :----- | :------------------ |
| 400    | 未指定source_id参数 |
| 404    | 未找到指定的视频流  |

------

## 接口6：停止所有视频流

| 项目 | 内容                     |
| :--- | :----------------------- |
| 接口 | `GET /stop_all`          |
| 描述 | 停止所有正在处理的视频流 |

**响应参数：**

| 参数    | 类型   | 说明         |
| :------ | :----- | :----------- |
| message | string | 操作结果信息 |

------

## 错误码说明

| 状态码 | 说明           |
| :----- | :------------- |
| 200    | 成功           |
| 400    | 请求参数错误   |
| 404    | 资源不存在     |
| 500    | 服务器内部错误 |
